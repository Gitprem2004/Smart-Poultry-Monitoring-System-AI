# 🐔 Smart Poultry Monitoring System using AI

## 🚀 Overview

An AI-powered poultry monitoring system that automates chicken detection, tracking, behavior analysis, and disease prediction using deep learning and computer vision.

This system reduces manual effort, improves efficiency, and enables early detection of health issues in poultry farms.

---

## 🎯 Key Features

* ✅ Real-time chicken detection using YOLOv8
* ✅ Multi-object tracking using DeepSORT
* ✅ Behavior classification (walking, eating, resting, inactive)
* ✅ Disease prediction using ResNet
* ✅ AI-based treatment suggestions
* ✅ Chatbot integration for intelligent assistance
* ✅ Supports image, video, and live camera input

---

## 🧠 System Pipeline

```
Input (Image/Video/Live Feed)
        ↓
YOLOv8 Detection
        ↓
DeepSORT Tracking
        ↓
ResNet Classification
        ↓
Health Prediction
        ↓
AI Treatment Suggestion (Chatbot)
```

```bash
pip install -r requirements.txt

```
